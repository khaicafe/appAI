Cấu hình key

OpenRouter key Chưa có
Gemini key Chưa có
Telegram bot token Chưa có
Nhập một trong hai key để cài. Nếu nhập key này app sẽ cấu hình OpenRouter với model mặc định là openrouter/qwen/qwen3.6-plus:free.
Hoặc nhập key này để app cấu hình Google Gemini với model mặc định là gemini-2.5-flash.
Nếu có token, app sẽ bật `channels.telegram.enabled` và lưu `channels.telegram.botToken` vào OpenClaw.

Hướng dẫn kết nối bot

Tab này dành cho người mới. Chỉ cần làm lần lượt từ trên xuống dưới, không cần biết OpenClaw hoạt động bên trong như thế nào.

Bước 1: Chuẩn bị key

Bạn cần ít nhất một key AI. Dễ nhất là dùng OpenRouter API key hoặc Gemini API key. Nếu muốn dùng Telegram bot thì vào BotFather để tạo một bot mới, sau đó BotFather sẽ trả về cho bạn một Telegram bot token và username của bot, ví dụ `@ten_bot_cua_ban`.

Bước 2: Nhập key vào tab Cấu hình key

Mở tab Cấu hình key, dán OpenRouter key hoặc Gemini key vào ô tương ứng. Nếu có Telegram bot token thì dán vào ô Telegram bot token. Sau đó bấm Lưu key để app ghi lại các key này cho những lần mở sau.

Bước 3: Cài OpenClaw

Chuyển sang tab Controller rồi bấm Cài đặt OpenClaw. App sẽ tự cài OpenClaw, cấu hình model mặc định, lưu provider và bật Telegram channel nếu bạn đã nhập bot token.

Bước 4: Chạy Gateway

Sau khi cài xong, bấm Chạy Gateway. Khi thành công, phần trạng thái sẽ báo gateway đang chạy và hiện URL cùng PID. Lúc này OpenClaw đã sẵn sàng nhận kết nối.

Bước 5: Kết nối Telegram bot

Trong tab Controller, bấm Connect Telegram một lần để app tải danh sách pairing Telegram đang chờ. Pairing code không phải lấy trong app này trước, mà lấy từ bot Telegram của bạn. Ở đây “bot của bạn” nghĩa là bot Telegram bạn đã tự tạo ở Bước 1 bằng BotFather. Hãy mở Telegram, tìm đúng username của bot đó, ví dụ `@ten_bot_cua_ban`, rồi nhắn cho bot một tin như `/start` hoặc một câu chào. Bot sẽ hiện pairing code cho bạn. Hãy copy đúng pairing code đó, quay lại app, dán vào ô Telegram pairing code, rồi bấm lại Connect Telegram để xác nhận.

Bước 6: Nếu gặp lỗi

Nhìn xuống phần Nhật ký ở cuối app. Tất cả bước cài đặt, chạy gateway và connect Telegram đều được ghi lại ở đó. Nếu app báo thiếu key, quay lại tab Cấu hình key để nhập lại rồi thử lại.

Mẹo: nếu bạn không biết nên chọn key nào, hãy dùng key nào bạn đã có sẵn. App chỉ cần một trong hai key OpenRouter hoặc Gemini để cài và chạy.
